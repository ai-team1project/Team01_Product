### 상품 등록
1. 제품명 입력 받기
2. 재고 수량 입력 받기
3.  가격 입력 받기
4. 카테고리 입력받기
5. 등록일
    현재 날짜 

### 상품 목록 조회
1. 고유번호 조회
2. 상품명 조회
3. 카테고리 조회

   - 하의
   - 악세사리
   - 신발
   - 아우터

   - 행사 이름 설정
   - 할인 가격 설정
